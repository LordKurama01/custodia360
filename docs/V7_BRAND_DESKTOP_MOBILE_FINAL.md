# Custodia360 v7 - Pulida final de marca, desktop y mobile

## Objetivo
Dejar Custodia360 como Fase 1 vendible y operativa: control de bultos, guías, pases USD, cuenta corriente por cliente y WhatsApp generado desde datos.

## Implementado
- Identidad visual definitiva del Brand Kit: logo, wordmark, icono, favicons, PWA manifest y OG image.
- Desktop reordenado como mesa de control de oficina.
- Mobile conservado como app rápida con bottom nav: Seguimiento, Cuentas, Nueva, Guías, Más.
- Sidebar desktop reorganizada por prioridad real: Mesa de control, Cuentas, Guías, Clientes.
- Seguimiento desktop convertido a tabla operativa compacta.
- Panel derecho de ficha rápida para operación seleccionada / primera operación filtrada.
- Cuenta corriente por cliente: pases pendientes, guías a reintegrar, movimientos especiales e historial de pagos.
- Cobro seleccionando pases.
- Movimiento especial para casos raros: proveedor no acepta USDT, adelanto de Jeremías, devolución, mercadería agotada o aplicar a otra compra.
- Landing/login con copy nuevo enfocado en control real, cuenta corriente y WhatsApp.
- Portal cliente pulido con marca y copy más simple.

## Criterio de arquitectura
Mobile no intenta ser desktop chico. Es operación rápida.
Desktop no intenta ser mobile grande. Es mesa de control.

## Alcance mantenido
No se agregaron módulos fuera de Fase 1. Todo lo nuevo corresponde a control de bultos, guías, pases, pagos simples y cuenta corriente.
