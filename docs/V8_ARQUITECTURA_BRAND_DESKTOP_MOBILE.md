# Custodia360 v8 — arquitectura visual y operativa

## Objetivo

Reordenar Custodia360 como sistema de logística operativo, separando claramente:

- Mobile: app rápida para operar en movimiento.
- Desktop: mesa de control para oficina.
- Marca: isologotipo separado de tipografía, sin depender del wordmark recortado.

## Cambios de marca

- Se separó el símbolo/isotipo de la tipografía.
- La marca se renderiza con texto vivo: `Custodia` en blanco + `360` en verde flúor.
- El símbolo usa el ícono transparente del brand kit.
- Se evita usar el logo completo como imagen en zonas chicas para que no se corte ni pierda legibilidad.

## Cambios desktop

- Sidebar más limpia y jerarquizada.
- Header superior más corto.
- Mesa de control con tabla operativa.
- Ficha rápida lateral como panel de lectura, no como card gigante.
- Selección de operación en tabla antes de abrir ficha completa.
- Botones principales: Ver, Cobrar, WhatsApp.
- Detalle corregido para evitar textos pegados como `ProveedorGénesis` o `Bultos5`.

## Cambios mobile

- Se mantiene lógica de app rápida.
- Bottom nav sigue operativo.
- Formularios quedan en paneles, no mezclados con seguimiento.
- Marca más limpia y no invasiva.

## Flujo recomendado

Desktop:

1. Mesa de control.
2. Buscar cliente/guía/proveedor.
3. Seleccionar operación.
4. Ver ficha rápida lateral.
5. Cobrar, cargar guía o mandar WhatsApp.

Mobile:

1. Seguimiento.
2. Cuentas.
3. Botón central para carga.
4. Guías.
5. Más.
