# Monetización v19

Custodia360 se vende como sistema privado de operación logística y cobros.

## Producto base

- Mesa operativa.
- Contactos: clientes y proveedores.
- Cobros vivos.
- Guías activas.
- Visor cliente.
- Multi-dueño controlado.

## Upsells

- Dueño adicional.
- Setup de tenant nuevo.
- Configuración de WhatsApp por dueño.
- Dominio propio.
- Marca blanca.
- Soporte mensual.
- Carga inicial de clientes/proveedores.

## Regla comercial

Jeremías puede tener límite inicial de espacios. Más dueños/espacios se habilitan como venta adicional de Prestige.
