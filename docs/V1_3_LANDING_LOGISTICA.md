# Custodia360 V1.3 — Landing logística + ingreso único

Cambio aplicado:

- `/login` deja de parecer selector de roles.
- Se transforma en landing pública simple, institucional y diferente al resto de proyectos.
- No habla de precios, planes, demo ni MVP.
- Habla de logística, transporte, seguridad operativa, trazabilidad y control.
- Mantiene un ingreso único visible: `Ingresar`.
- Mantiene WhatsApp configurable por variables de entorno.
- El usuario final no elige rol. El rol se resolverá por autenticación/permisos.

Rutas internas de desarrollo se mantienen intactas.
