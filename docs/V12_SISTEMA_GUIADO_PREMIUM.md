# Custodia360 v12 — Sistema guiado premium

Objetivo: dejar Fase 1 más clara, más rápida y con menos sensación de página estática.

## Cambios clave

- Visor cliente premium: estado único, fecha/hora, historial oculto.
- No se muestran estados internos al cliente.
- Guías como documentos privados clickeables.
- Command palette (`Ctrl + K`) en mesa de control.
- Sidebar: Fase 1 activa y resto en construcción.
- Footer Prestige global con diamante dorado.
- Bottom nav mobile separado del footer.

## Regla de producto

Custodia360 debe ser más rápido que marcar a mano. Cada carga debe estar guiada y no debe exigir formularios largos si no hace falta.
